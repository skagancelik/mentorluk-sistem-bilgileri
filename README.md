Bu repository, Mentorluk Programı Yönetim Sistemi'nin sistem tipolojisi ve teknik bilgilerini içeren HTML dokümantasyonunu barındırır.

## GitHub Pages

Bu sayfa GitHub Pages üzerinden yayınlanmaktadır:
**https://skagancelik.github.io/mentorluk-sistem-bilgileri/**

## İçerik

- Sistem mimarisi ve tipolojisi
- IT & Güvenlik ekipleri için Q&A
- Frontend, Backend, Deployment detayları
- Entegrasyonlar (Zoom, Microsoft, Google Calendar)

## GitHub Pages Kurulumu

1. Repository Settings > Pages
2. Source: `Deploy from a branch`
3. Branch: `main`
4. Folder: `/ (root)`
5. Save

Sayfa birkaç dakika içinde yayınlanır.
